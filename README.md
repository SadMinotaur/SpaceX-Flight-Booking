# React-Redux-Saga-Template

Based on https://github.com/Revolt9k/react-redux-template.

#### This template uses:

- React
- Redux
- Redux-Saga
- TypeScript
- Scss
- Airbnb eslint setup and basic prettier config
- Yarn PnP
- Husky

#### Webpack build includes:

- Aliases
- Chunk splitting
- Hot reload
- Analyzing bundle size tool
